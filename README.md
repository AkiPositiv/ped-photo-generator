# Ped Photo Generator

Generate AI photos for each pedagogical workload item using Fal.ai PhotoMaker.

## Live app

[username.github.io/ped-photo-generator](https://username.github.io/ped-photo-generator)

## Setup

1. Get a Fal.ai API key at [fal.ai/dashboard](https://fal.ai/dashboard)
2. Open the app
3. Enter your API key (saved locally in browser)
4. Upload your педнагрузка Excel file (generated by the main ped_yuklama tool)
5. Select month, mode, upload teacher photos, and generate

## How it works

- **Excel parsing**: reads plan/actual hours from monthly sheets (S, O, N, D, Y, F, M, A, May, Iyun)
- **Photo generation**: Fal.ai PhotoMaker model with multiple face reference images
- **Angle variety**: 10 random angle/composition variations across work items

## Local development

Open `index.html` directly in a browser — no server needed.
Run tests by opening `tests/test.html` in a browser.
